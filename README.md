# 🤖 Identification of AI-Generated Text

A Natural Language Processing (NLP) project that automatically identifies whether a given text is generated by an AI model or written by a human, using fine-tuned transformer models (BERT and DeBERTa-v3).

---

## 📁 Project Structure

```
AI_text_detection/
├── data/
│   ├── raw/              # Original downloaded datasets
│   └── processed/        # Cleaned and tokenized data
├── notebooks/
│   ├── 01_EDA.ipynb                  # Exploratory Data Analysis
│   ├── 02_Preprocessing.ipynb        # Data cleaning and tokenization
│   ├── 03_Training_BERT.ipynb        # BERT fine-tuning (baseline)
│   └── 04_Training_DeBERTa.ipynb     # DeBERTa-v3 fine-tuning (primary/SOTA)
├── src/
│   ├── preprocess.py     # Data preprocessing utilities
│   ├── model.py          # Model loading and configuration
│   ├── train.py          # Training pipeline
│   └── evaluate.py       # Evaluation and metrics
├── models/               # Saved trained models
├── results/              # Evaluation results, plots
├── reports/              # Final report and documentation
├── requirements.txt      # Python dependencies
└── README.md
```

## 🤗 HuggingFace Ecosystem

This project is fully built on the HuggingFace ecosystem:

| Package | Role | Used For |
|---|---|---|
| `huggingface-hub` | Connect to HuggingFace Hub | Auto-installed with `transformers` |
| `transformers` | Load BERT, DeBERTa models + tokenizers | Model loading & fine-tuning |
| `datasets` | Stream & load HC3 dataset | One-line dataset access |
| `tokenizers` | Fast tokenization backend | Text preprocessing |
| `evaluate` | Compute standard NLP metrics | F1, Accuracy calculation |
| `accelerate` | Optimized training on CPU/GPU | Training pipeline |

```python
# Load dataset directly from HuggingFace Hub
from datasets import load_dataset
dataset = load_dataset("Hello-SimpleAI/HC3", "all")

# Load DeBERTa model from HuggingFace Hub
from transformers import AutoModelForSequenceClassification
model = AutoModelForSequenceClassification.from_pretrained("microsoft/deberta-v3-base")
```
